📡 API ENDPOINTS TERSEDIA:

FOODS:
   GET    /api/foods
   GET    /api/foods/:id
   POST   /api/foods
   PUT    /api/foods/:id
   DELETE /api/foods/:id

CATEGORIES:
   GET    /api/categories

REVIEWS:
   GET    /api/reviews/:foodId
   POST   /api/reviews

FAVORITES:
   GET    /api/favorites/:userId
   POST   /api/favorites
   DELETE /api/favorites/:userId/:foodId

ORDERS:
   GET    /api/orders/:userId
   POST   /api/orders

❓ TROUBLESHOOTING:

Backend tidak mau jalan?
   → Cek MySQL running
   → Cek port 5000 available
   → Check .env file correct
   → Run npm install di express folder

Frontend blank/error?
   → Buka DevTools (F12)
   → Check console untuk error message
   → Pastikan backend running di 5000
   → Run npm install di react folder

CORS error?
   → Sudah dikonfigurasi di server.js
   → Pastikan backend running di port 5000
